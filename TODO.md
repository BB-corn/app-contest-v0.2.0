## 常见问题

**Q: 代码冲突怎么办？**
A: 每人负责不同文件，尽量避免冲突。必须改同一文件时，先沟通。

**Q: 搞不定怎么办？**
A: 立即降级！先做简单版，保证能用，再优化。

**Q: 高德 API 失效？**
A: 项目已内置 fallback，会自动用平滑曲线。检查 `.env` 配置。
